API

  Geolocation API
  위치기반을 사용하면 자신의 위치를 웹 어플리케이션에 제공할 수 있음
  coordinates : 지리적 위치 좌표
  .latitude,longitude 경도
  .altitude고도,accuracy정확성
  position : 현재위치를 정의하는 coordinates객체를 리턴
  getCurrentPosition() : 장치의 현재 위치를 리턴

  gps가 있는 장치에서 가장 정확함
  사용자 위치 가져오기
  
  

  
  
Console

  
  method
  assert() : 콘솔에 메세지로 작성
  clear() : 콘솔에서 모든 메세지를 지움
  count() : 반복문처럼 콘솔을 지정한 범위만큼 호출함
  error() : 콘솔에 오류 메세지를 쓰임
  group() : 메세지 그룹을 생성함
  groupCollapsed() : 축소된 메세지 그룹을 만듬(접었다 폈다)
  groupEnd() : 메세지 그룹을 종료함
  info() : 콘솔에 메세지를씀
  log() : 콘솔에 메세지를 기록함
  table() : 콘솔에서 배열을 테이블로 작성함
  time() : 콘솔에서 지정된 명령이 걸린시간을 표현함
  timeEnd() : 타이머를 종료하고 결과를 콘솔에 기록함
  trace() : 특정지점에서 어떻게 끝났는지 보여주는 추적을 표시함
  warn() : 콘솔에 경고를 씀
