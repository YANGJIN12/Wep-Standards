array

    - 배열을 쓰는 이유 > 단일변수에 여러값을 저장하고자 할 때

배열매서드 및 속성

   - concat() : 두 배열끼리 결합을 시켜줌 
   - constructor : array 프로토타입 만든 함수값을 리턴 {[]}
   - copyWithin() : 배열에 요소를 배열의 다른 위치에 복사함
   - entries() : 배열에서 키[indexing]와 값[value]을 쌍으로 반복
   - every() : 모든 값에 대해서 확인
   - fill() : 배열에 모든 요소를 덮어 씌움
   - filter() : 이상에 값을 리턴
