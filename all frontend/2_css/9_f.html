  flex는 아래에 대한 속기속성
  - flex-grow : 성장 
  - flex-shrink : 수축
  - flex-basis : 기반
  
  flex에 목적 > 가로나 세로정렬을 이름 그대로 유연하게 해줌
  flex를 사용하기전에 display 변천사
  table > float > flex,grid
  table : 가로로 정렬되지만 변형이 어려움 > div 코딩 단점[가로정렬을 css로 반듯이 설정]
  float left,right > overflow 되면 레이아웃에 이상이 생김
  flex : 엘리먼트가 오버플로우 되도 가로로 강제 정렬됨
