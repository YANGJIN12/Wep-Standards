Text Formatting

  특별한 의미가 있는 텍스트를 정의하기 위한 요소가 포멧팅에 포함되어 있음
  - b : 굵은 텍스트
  - strong : 굵은 텍스트
  - i : 기울림꼴 텍스트
  - em : 강조된 텍스트
  - mark : css를 쓰지않고 유일하게 배경색이 나오는 텍스트
  - small : 부모를 기준으로 크기를 80%로 표시
  - del : 문서에서 삭제된 텍스트를 정의
  - ins : 문서에서 삽입된 텍스트를 정의
  - sub : 아래 첨자 
  - sup : 위 첨자
